# DeepFaceCraft – Human Face Generation using DCGAN

## Project Description
DeepFaceCraft is a deep learning based system that generates realistic human faces using Deep Convolutional Generative Adversarial Networks (DCGAN).

## Technologies Used
- Python
- TensorFlow / Keras
- Django
- HTML, CSS
- MySQL

## Steps to Run the Project

1. Clone the repository
git clone https://github.com/yourusername/DeepFaceCraft.git

2. Navigate to project folder
cd DeepFaceCraft

3. Install required packages
pip install -r requirements.txt

4. Run the project
python manage.py runserver

5. Open browser and go to
http://127.0.0.1:8000/

## Project Features
- User Login and Signup
- Face generation using DCGAN
- Result visualization

## Results
The system generates synthetic human faces after training the DCGAN model.
